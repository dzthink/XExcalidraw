# Excalidraw File Formats

## Primary Write Format

- `*.excalidraw`
- Content: Excalidraw scene JSON.
- No private or custom fields allowed.

## Compatible Read/Import Formats (Read-only)

- `*.excalidraw.json`
- `*.excalidraw.svg`
- `*.excalidraw.png`

> Note: embedded scene data in SVG/PNG is for import/export only and must not replace the primary `.excalidraw` file.
